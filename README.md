# ネットワークプログラミング課題 Webページ

## 📌 概要

本プロジェクトは、ネットワークプログラミングの授業課題として作成したWebページです。
C言語の各問題（A-1, A-2, B, C）の解答や、学習内容をまとめたサイトになっています。

## 🎯 目的

* C言語の理解を深める
* Webページ作成の基礎（HTML）を習得する
* 課題内容を整理し、見やすくまとめる

## 🛠 使用技術

* HTML
* C言語
* Git / GitHub

## 📂 コンテンツ

* C言語 問題A-1の解答
* C言語 問題A-2の解答
* C言語 問題Bの解答
* C言語 問題Cの解答
* Webページ作成時に注意した点
* C言語の特徴

## 🚀 使い方

1. このリポジトリをクローン

```bash
git clone https://github.com/ユーザー名/リポジトリ名.git
```

2. index.html をブラウザで開く

## 💡 工夫した点

* 各課題にすぐアクセスできるようにメニュー形式で構成

* 学習内容（C言語の特徴など）もまとめて理解を深めた
