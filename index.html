[問題]

次の行列の積に関するプログラムで使われるメソッドを JAVA 言語で作成せよ.

このプログラムは, 要素として int 型の値を持つ 3 × 3 行列に関する操作を行っ ている.

2 つのメソッド, メソッド matAvrDiagElements とメソッド mat4thPower のみ作成せよ. メソッド matProduct は作成済みとする.

メソッド matAvrDiagElements は, 第 1 引数の行列からその行列の対角要素 (3 個の要素) の平均値を求めて返り値として返す.

メソッド matProduct は, 第 1 引数の行列 (A) と第 2 引数の行列 (B) の積 (AB) の結果を第 3 引数の行列に格納する.

メソッド mat4thPower は, 第 1 引数の行列を 4 乗した結果を第 2 引数の行列に格納する. なお, 必要であれば, これら 3 つのメソッドはお互いに内部で使用しても構わない.

ただし, 作成するメソッドはメインメソッド内に示されている行列の各要素の値が変わっても, 正しく動作するように作成すること.

 37 /* --- 対角成分の平均値 --- */
 38 double matAvrDiagElements(int a[3][3]) {
 39     double sum = 0.0;
 40     int i;
 41     for (i = 0; i < 3; i++) {
 42         sum += a[i][i];
 43     }
 44     return sum / 3.0;
 45 }

 48 /* --- 行列積 C = A × B --- */
 49 void matProduct(int a[3][3], int b[3][3], int c[3][3]) {
 50     int i, j, k;
 51     for (i = 0; i < 3; i++) {
 52         for (j = 0; j < 3; j++) {
 53             c[i][j] = 0;
 54             for (k = 0; k < 3; k++) {
 55                 c[i][j] += a[i][k] * b[k][j];
 56             }
 57         }
 58     }
 59 }

 62 /* --- 4乗行列 D = A^4 --- */
 63 void mat4thPower(int a[3][3], int d[3][3]) {
 64     int temp1[3][3];
 65     matProduct(a, a, temp1);   // A^2
 66     matProduct(temp1, temp1, d); // (A^2)^2 = A^4
 67 }
